# xpath2css
The simplest XPATH expression to CSS/jQuery selector conversion tool.

## Install
```
$ npm install --save xpath2css
```

## Usage
```js
var xpath2css = require('xpath2css');

console.log(xpath2css(process.argv[2]));
```
